# djpics
простое приложение на джанго 2.0 - каталог хранения изображений.

Возможности:

* Добавление изображения в каталог;
* Удаление изображения;
* Поиск изображения по атрибутам;
* Изменение атрибутов изображения:
  * Название и описание;
  * Дата съёмки.
